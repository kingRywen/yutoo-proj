## 安装

```shell
npm install @yutoo/components -S
# or
yarn add @yutoo/components
```

## Quick Start

```javascript
import Vue from 'vue'
import YutooUI from '@yutoo/components'

Vue.use(YutooUI)

// or
import {
  YtTransition
  // ...
} from '@yutoo/components'

Vue.component(YtTransition.name, YtTransition)
```
