# 宇图快速搭建前端模板插件

适用于[vue-cli@3.0](https://github.com/vuejs/vue-cli)快速开发的前端框架

### 安装

如果尚未安装 vue-cli 3，请先按照此处的安装说明进行操作： https://github.com/vuejs/vue-cli

用 vue-cli 3 生成一个项目

```cmd
vue create <project-name>
```

在安装插件之前，先删除 src 中的文件，请确保提交了你的代码，以免造成意外的损失，安装时可能会修改项目中任意现存的文件

安装插件

```cmd
cd <project-name>
vue add yutoo-template
```
